# Watering WebApp

A web application for the soil monitoring and prescriptive irrigation system **SMARTER**. A precision farming technology for orchards smart irrigation. 

### Architecture

The project is developed using `Node.js` for the backend and `Vue 3` for frontend.

![architecture](https://github.com/ManuelePasini/watering_web_refactoring/assets/62949013/39353053-77f0-4c7c-8f8b-03da535f67de)

### Deploy
The project can be build and deployed using `Docker` and the repository contains the script for building docker images and for the deployment using `docker-compose`.

```sh
./deploy.sh
```
